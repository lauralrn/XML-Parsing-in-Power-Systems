# EH2745-Assigment_1
Solution of Assignment 1 in EH2745
Laura Laringe, 30.04.20


The folder "Classes" contains the classes created.
The program reads the xml selected from the GUI.
TopologyGenerator.py file is used to find the topology of the grid. In output it has two lits: one representing all the CEs in order and one representing all the nodes of the grid in order.
Main.py represent the rest of the code.
THE GUI will guide through the steps of the topology algorithm, with an option to stop. Furthermore it will give the user the possibility to choose one node and display its information.

pip install the requirement.txt before running
